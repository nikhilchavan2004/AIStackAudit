# Prompts

## Purpose

AI prompts are used to generate optimization summaries and intelligent recommendations for users based on their AI SaaS usage patterns.

---

# Prompt Inputs

The following data is considered:
- AI tool selected
- current subscription plan
- monthly spending
- team size
- number of seats
- primary use case

---

# Example Prompt Logic

The backend generates contextual summaries such as:
- recommending lower-cost plans
- reducing unused licenses
- optimizing team subscriptions
- consolidating AI tools

---

# Goal

The goal of the prompt system is to simulate AI-powered consulting recommendations for SaaS cost optimization.
